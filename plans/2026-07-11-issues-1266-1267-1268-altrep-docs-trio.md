## Compile-verification procedure (every Rust block you touch)

1. Create `rpkg/src/rust/docs_compile_probe.rs` containing each rewritten
   example (rename types per block to avoid collisions), and add
   `mod docs_compile_probe;` to `rpkg/src/rust/lib.rs`.
2. `cd rpkg/src/rust && cargo check 2>&1 > /tmp/docs-trio-check.log` — cwd
   matters: from the repo root, `.cargo/config.toml` patches are missed and
   you get ~30 phantom errors against git-sourced crates. Read the log.
3. Fix examples until clean, copy the FINAL text back into the docs,
   then **delete the probe file and the `mod` line — they must NOT be
   committed** (`git status` must show only `docs/*.md` changes).
4. Note in the PR body that every block was compile-verified this way.
